## 개요

### 목표
- **단일 화면 중심**: 파일을 열면 즉시 마크다운 콘텐츠에 집중
- **직관적인 아이콘 기반 UI**: 텍스트 대신 아이콘으로 기능 표시
- **편안한 사용자 경험**: 최소한의 클릭으로 작업 완료
- **국제적 접근성**: 언어에 관계없이 이해 가능한 UI

### 디자인 철학
- **Minimalism**: 불필요한 요소 제거
- **Content First**: 콘텐츠에 집중
- **Intuition**: 직관적인 인터페이스
- **Consistency**: 일관된 디자인 패턴

## 디자인 원칙

### 1. 단일 화면 중심
- 파일을 열면 즉시 마크다운 콘텐츠 표시
- 파일 탐색기는 필요 시에만 표시
- 편집 모드는 전체 화면으로 전환

### 2. 아이콘 기반 UI
- Font Awesome 아이콘 사용
- 모든 기능을 아이콘으로 표시
- 툴팁으로 기능 설명 제공

### 3. 블랙 앤 화이트 테마
- 깔끔한 흑백 디자인
- 색상은 최소한으로 사용 (상태 표시용)
- 고대비로 가독성 향상

### 4. 반응형 디자인
- 데스크톱, 태블릿, 모바일 지원
- 화면 크기에 따라 레이아웃 조정
- 터치 친화적 인터페이스

## 컴포넌트 설계

### 1. Header (헤더)

#### 구성 요소
```
┌──────────────────────────────────────────────────┐
│ [☰] Document.md        [✏️] [💾] [⚙️] [👤]     │
└──────────────────────────────────────────────────┘
```

**요소 설명:**
- `[☰]` (햄버거 메뉴): 사이드바 토글
- `Document.md`: 현재 파일명
- `[✏️]` (편집 아이콘): 편집 모드 진입
- `[💾]` (저장 아이콘): 파일 저장 (편집 모드에서만 표시)
- `[⚙️]` (설정 아이콘): 설정 메뉴
- `[👤]` (사용자 아이콘): 사용자 메뉴 (로그아웃 등)

**상태 표시:**
- 저장 상태: `[💾]` 옆에 텍스트 표시
  - "저장됨" (회색)
  - "저장 중..." (파란색)
  - "저장 실패" (빨간색)

**스타일:**
```css
.header {
  height: 60px;
  background: #ffffff;
  border-bottom: 1px solid #e0e0e0;
  display: flex;
  align-items: center;
  padding: 0 20px;
  position: fixed;
  top: 0;
  width: 100%;
  z-index: 1000;
}
```

---

### 2. Sidebar (사이드바)

#### 구성 요소
```
┌─────────────────────┐
│  파일 관리          │
├─────────────────────┤
│ 📁 folder1         │
│   📄 file1.md      │
│   📄 file2.md      │
│ 📄 document.md     │
│ 📄 notes.md        │
├─────────────────────┤
│ [➕ 새 파일]       │
│ [📤 업로드]        │
└─────────────────────┘
```

**기능:**
- 폴더 트리 구조 표시
- 파일 클릭 시 즉시 로드
- 새 파일 생성
- 파일 업로드
- 검색 기능

**위치:**
- 기본: 왼쪽 (숨김)
- 토글 가능
- 위치 변경 가능 (왼쪽/오른쪽)

**스타일:**
```css
.sidebar {
  width: 250px;
  background: #f5f5f5;
  border-right: 1px solid #e0e0e0;
  height: calc(100vh - 60px);
  overflow-y: auto;
  position: fixed;
  left: 0;
  top: 60px;
  transition: transform 0.3s ease;
}

.sidebar.hidden {
  transform: translateX(-100%);
}
```

### 3. Main Content Area (메인 콘텐츠 영역)

#### 뷰어 모드
```
┌─────────────────────────────────────────────┐
│                                             │
│         (여백)                              │
│                                             │
│    ┌─────────────────────────────────┐     │
│    │                                 │     │
│    │    # Document Title             │     │
│    │                                 │     │
│    │    Content here...              │     │
│    │                                 │     │
│    │    - List item 1                │     │
│    │    - List item 2                │     │
│    │                                 │     │
│    └─────────────────────────────────┘     │
│                                             │
│         (여백)                              │
│                                             │
└─────────────────────────────────────────────┘
```

**특징:**
- 중앙 정렬
- 최대 너비 제한 (예: 800px)
- 가독성 최적화
- 코드 하이라이팅 지원

**스타일:**
```css
.main-content {
  max-width: 800px;
  margin: 0 auto;
  padding: 40px 20px;
  line-height: 1.6;
  font-size: 16px;
}
```

│ │ # Document Title                      │  │

│ │ Content here...                       │  │
│ │                                       │  │

**특징:**

## 사용자 흐름

### 1. 로그인 흐름

```
[로그인 페이지]
    │
    ▼
[Google 로그인 버튼 클릭]
    │
    ▼
[Google 인증]
    │
    ▼
[메인 페이지로 리다이렉트]
    │
    ▼
[마지막 문서 자동 로드]
    │
    ▼
[뷰어 화면 표시]
```

**UI 상태:**
- 로그인 전: 로그인 페이지만 표시
- 로그인 후: 헤더에 사용자 아이콘 표시

## 반응형 디자인

### 브레이크포인트

```css
/* 모바일 */
@media (max-width: 768px) {
  .sidebar {
    width: 100%;
    transform: translateX(-100%);
  }
  
  .main-content {
    padding: 20px 10px;
  }
}

/* 태블릿 */
@media (min-width: 769px) and (max-width: 1024px) {
  .sidebar {
    width: 200px;
  }
}

/* 데스크톱 */
@media (min-width: 1025px) {
  .sidebar {
    width: 250px;
  }
}
```

### 모바일 최적화
- 햄버거 메뉴로 사이드바 토글
- 터치 친화적 버튼 크기 (최소 44x44px)
- 스와이프 제스처 지원

## 접근성

### 키보드 네비게이션
- `Tab`: 다음 요소로 이동
- `Shift + Tab`: 이전 요소로 이동
- `Enter`: 버튼/링크 활성화
- `Esc`: 모달/사이드바 닫기
- `Ctrl + S`: 저장

### ARIA 레이블
```html
<button aria-label="파일 저장">
  <i class="fas fa-save"></i>
</button>

<nav aria-label="파일 탐색기">
  <!-- 파일 목록 -->
</nav>
```

### 스크린 리더 지원
- 모든 아이콘 버튼에 텍스트 레이블 제공
- 상태 변경 시 알림 제공
- 포커스 관리
